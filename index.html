uses a devices hotspot to share files. used uploadserver, python and chatgpt for the webpage html becasue i cant think for myself. 1. run hotspot - nmcli connection up [hotspot name] 2. cd into whatever you want to share from/to 3. run python3 -m uploadserver 4. connect by using localhost/0.0.0.0:8000 on main machine 4.5 connect by using devices IP when on hotspot. 5. for downloading files you can just use python3 -m http.server 8000 6. to kill, turn off hotspot in settings and run lsof -i 8000 6.5 run kill [whatever the PIP is] theses notes are reaaally poorly written but if you need help ill try my best to assist. i didnt actually make anything here if it wasnt clear earlier
